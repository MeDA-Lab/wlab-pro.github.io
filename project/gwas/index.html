GWAS

Genome-Wide Assiciated Studies

GWAS project deals with large scale GWAS problem which contains tens of billions of linear regression and t-test. We have written a parallel MPI-GPU code which can be executed on multiple GPU clusters. We achieve thousands of times faster performance which make extreme large GWAS problems become practical.
